msoto App

Apply for a soft loan based on electronic.

Create Product

You can create a product through this rest API application through the endpoint /products and the POST verb. You must be a logged in user to access this role. This is accomplished using a jsonwebtoken

Edit Product

This role is also to only the logged in users. JWT is also used here to verify if the user is authorized to use this resource /products PATCH verb

Get Products

This is not a protected route and so any body can view it. To get all products, the id property is not used. For a single product, the product with the specific id is fetched. /products or /products/:id and GET verb

Delete Product

endpoint /products/:id and DELETE verb. It is a protected route and anyone who wants to access it must have a valid JWT. It deletes a product with the specific id

Monitor your product

Use postman to make requests to the various endpoints with the appropriate verbs. It also has the orders resouce with /orders endpoint. You can use the same verbs as above
